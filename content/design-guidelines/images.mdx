## Adding images

Images can be added to any presentation by dragging and dropping, copy and pasting, or inserting from the toolbar in Keynote, Google Slides, and Figma.

### Adding images in Keynote

|    Source     |                        Description                        |                                                                                                 Example                                                                                                 |
| :-----------: | :-------------------------------------------------------: | :-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|    Browser    | Drag and drop any image from the web browser into Keynote |     ![animated gif example of dragging and dropping an image from a web browser into keynote](https://user-images.githubusercontent.com/10384315/55426909-c4385100-553a-11e9-88b6-467a4134f1cf.gif)     |
| Local machine |   Drag and drop any image from the Finder into Keynote    | ![animated gif example of dragging and dropping an image from a the Finder in macOS into keynote](https://user-images.githubusercontent.com/10384315/55426910-c4385100-553a-11e9-811a-69ecc5710708.gif) |
