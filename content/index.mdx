Primer Presentations is the presentation design system for GitHub. This repo contains the presentation templates, guidelines, and assets needed to create presentations that represent the GitHub brand.

<Note variant="warning">


The Primer presentation design system is intended to be used _**only**_ by GitHub staff for presentations that
represent GitHub.

</Note>
